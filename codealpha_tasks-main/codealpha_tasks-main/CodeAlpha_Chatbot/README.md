📜 Project Overview

This is a rule-based chatbot developed as part of the CodeAlpha Python Programming Internship.
The chatbot interacts with users through simple text-based commands and responds with predefined replies.
It demonstrates the use of conditional statements, loops, and string manipulation in Python.

🧠 Key Python Concepts Used

if-elif conditions

Loops (while True)

String handling (lower(), strip())

Functions

Basic user input/output

⚙️ How It Works

The chatbot greets the user.

It listens for text input (like “hello,” “how are you,” or “bye”).

Based on input, it responds using predefined rules.

The conversation ends when the user types “bye.”

To run the chatbot:

python chatbot.py
